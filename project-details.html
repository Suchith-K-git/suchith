Automated Web App Deployment: Integrating Jenkins, GitHub, and Apache Tomcat on EC2

This project involves setting up an automated deployment pipeline using Jenkins, GitHub, Maven, and Apache Tomcat on an EC2 instance. The setup allows for seamless code updates and application deployments.

The workflow begins with a developer pushing code to GitHub, which Jenkins then pulls and builds using Maven. The resulting WAR file is deployed to Tomcat on an EC2 instance, where users can access the web application.
